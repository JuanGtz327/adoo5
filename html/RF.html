Requerimientos Funcionales

  • RF_1. RF1. Registrar Producto. El sistema debe permitir agregar productos al catálogo de acuerdo a la categoría del producto, para añadir un producto se deberá de escribir su nombre, tipo de producto, código del producto (sku o id), precio, categoría, descripción y añadir 3 imágenes del producto.
  • RF_2. RF2.Registrar Venta. El sistema debe registrar los datos de una venta fecha, producto o productos, monto.
  • RF_3. RF3.Pagar Producto. El sistema debe de permitir realizar un pago de compra, este pago puede ser un abono parcial o completo.
  • RF_4. RF4. Mostrar Catálogo. El sistema debe de permitir mostrar el catálogo de productos en el inicio de la página y permitir compartirlo.
  • RF_5. RF5. Contactar al Administrador. El sistema debe permitir mandar un mensaje directo al administrador.
  • RF_6. RF6.Responder feedback. El sistema debe de permitir al administrador los mensajes que recibe de los clientes.
  • RF_7. RF7.Compartir información de producto en redes sociales. El sistema debe permitir compartir un producto o catalogo en redes sociales.
  • RF_8. RF8.Visualizar comparación de ventas. El sistema debe comparar las ventas a traves de los datos del registro de cada venta.
  • RF_9. RF9.Inicio de Sesión. El sistema debe contar con la sección de inicio de sesión para que el usuario pueda acceder a su historial de compras, catálogo, pedidos y la opción de contactar al administrador.
